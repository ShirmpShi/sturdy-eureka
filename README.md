# Sturdy Eureka

## Overview
Sturdy Eureka is a Telegram bot designed to assist users with various tasks, providing a simple and efficient interface for interacting with the Telegram API.

## Features
- **User Commands:**
  - `/start`: Initializes the bot and provides a welcome message.
  - `/help`: Lists all available commands and how to use them.
  - `/info`: Provides information about the bot and its creator.

- **Custom Commands:**
  - Users can create custom commands tailored to their needs.

## Installation
To install the bot, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/ShirmpShi/sturdy-eureka.git
   ```
2. Navigate to the project directory:
   ```bash
   cd sturdy-eureka
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```
4. Create a `.env` file and add your Telegram bot token:
   ```plaintext
   TELEGRAM_TOKEN=your_telegram_token
   ```
5. Run the bot:
   ```bash
   npm start
   ```

## Usage
Once the bot is running, users can interact with it through the Telegram app by sending commands. The bot will respond based on the commands issued.
